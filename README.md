# Satellite Land Use Classification Framework

## Project Overview

This project develops a CNN-based framework for satellite land-use classification using deep learning and transfer learning approaches.

## Dataset

Dataset Used: EuroSAT
Dataset Link: https://www.kaggle.com/datasets/apollo2506/eurosat-dataset

## Models Used

* Custom CNN
* MobileNetV2
* ResNet50
* EfficientNetB0

## Installation

pip install -r requirements.txt

## Running the Project

1. Download dataset
2. Update dataset path
3. Run preprocessing notebook
4. Train models
5. Run evaluation notebook

## Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1 Score
* ROC-AUC
* Confusion Matrix
